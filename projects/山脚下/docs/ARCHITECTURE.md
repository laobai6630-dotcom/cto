# 系统架构

## 整体架构

山脚下项目采用模块化设计，包含以下核心模块：

### 1. 数据采集模块

- 日线/分钟线数据
- 周线/月线数据
- 财务基本面数据

### 2. 特征工程模块

- 134个原始特征提取
- 10个AI特征合成
- 10个筹码特征计算
- 特征标准化与选择

### 3. 对照组模块

- 识别跌幅前20名
- 提取对照组特征
- 对比分离度分析

### 4. ML训练模块

- 逻辑回归模型
- 随机森林模型
- 梯度提升模型
- 模型集成

### 5. 筛选模块

- 相似度计算
- 递进筛选逻辑

### 6. 跟踪报告模块

- 30天跟踪
- 日报/周报/月报

## 数据流

```
数据采集 → 特征提取 → ML模型 → 相似度筛选 → 候选股票 → 30天跟踪
    ↓
对照组识别 → 对照组特征 → 对比分析
```
